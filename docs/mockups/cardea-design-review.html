Design Critique — applied against updated spec
Cardea Design Review
The 3-tier gradient hierarchy is the system's strongest architectural idea — when followed correctly the gradient fires exactly once per screen, creating a clear focal point without visual noise. The animated ECG hero card is the single most memorable element and directly reinforces the app's coaching purpose. Key opportunities: enforce the tier rule more rigorously in the ring system, lift the zone-status pill to readable size, and fix two stale token values that leaked through the spec/code drift.
Usability
Zone pill is too small for in-motion reading
At 9–10sp the zone status pill is illegible with sweaty hands or a brief glance. This is the most safety-critical display element — zone changes require a prompt audio + visual response. Minimum 12sp, 40dp height touch target.
Selected preset card reads too subtly at distance
The gradient border on the selected PresetCard is hard to distinguish from the default glass border on a dim phone screen outdoors. Add a very subtle gradient-tinted background fill (GradientRed at ~6% alpha) to the selected state.
MaxHR field has no range validation signal
The AccountScreen MaxHR field accepts any integer. Values outside 100–220 will produce nonsensical preset targets silently. At minimum show a caption: "Typical range: 150–210 bpm."
Inactive nav label contrast fails WCAG AA
TextTertiary #52525B on background #050505 is ≈2.8:1 — below the 3.0:1 minimum for UI labels. Raise to #6B6B73 (≈3.5:1). One token change, global fix.
Visual Hierarchy
BootcampTile ring must not use CardeaNavGradient
If the ring progress arc uses the Blue→Cyan gradient, the Home screen has two gradient accent points (GoalTile + BootcampTile ring). This breaks the "one gradient per screen" rule. Use plain white arc at 0.55 opacity — it reads clearly and the ring shape does the work. See mockup above for correct rendering.
Coaching strip ECG icon too faint
0.5 opacity on a Tier 3 surface (which is itself 4% alpha on the dark bg) renders the icon nearly invisible. Lower practical limit is ~0.65 opacity.
Account screen Profile card gradient border is subtle
A 1.5dp gradient border on a 52px wide avatar card is the only Tier 1 signal on the Account screen. Consider pairing it with a very faint gradient wash inside the card (gradient at 4% alpha) to make the Tier 1 designation perceptible.
Consistency
Stale token: 0x0FFFFFFF in screen files
Old GlassBorder value (6% alpha). Canonical is 0x1AFFFFFF (10%). Search all .kt files for the old value and replace with the GlassBorder token.
Stale token: 0xFF9AA4B2 in screen files
Old CardeaTextSecondary (blue-tinted #9AA4B2). Canonical is #A1A1AA (neutral). Search for 0xFF9AA4B2 and replace with CardeaTextSecondary token.
All CTAs must use CardeaCtaGradient, not CardeaGradient
The full 4-stop gradient on a button reads as a rainbow. The CTA variant (Red→Pink) is warmer, more energetic, and specifically designed for call-to-action surfaces. Check all Button composables.
What Works Well
✓
3-tier hierarchy is the right architectural decision. A constraint that produces clarity rather than limiting creativity — the gradient fires as a highlight, not wallpaper.
✓
ECG hero card is the app's signature moment. The gradient trace directly maps to the product's purpose. Animated and layered with a glow — immediately distinctive.
✓
SegmentTimelineStrip is information-dense and elegant. 4dp of colored bar communicates the entire workout structure at a glance. The intensity color coding is intuitive.
✓
Three gradient variants prevent ambiguity. Full, CTA, Nav — each has a clear role. No designer needs to guess which gradient goes on which surface.
✓
Glass surface system is disciplined. Three properties (fill, border, highlight) create a consistent material at every elevation. No surface reinvents the language.
✓
Zone colors are semantically pure. Green/amber/red used only for zone status — never decoratively. Users learn the color language fast.
Priority Fix List — Ranked by Visual Impact
1
Remove gradient from BootcampTile ring — use white arc at 0.55 opacity
Violates one-gradient-per-screen. Instant fix, high impact on hierarchy clarity. ui/home/HomeScreen.kt · BootcampTile.
2
Switch all CTA buttons from CardeaGradient to CardeaCtaGradient
The full 4-stop gradient reads as a novelty on buttons. CardeaCtaGradient (Red→Pink) is energetic and purpose-built. All Button composables.
3
Audit and replace stale token literals 0x0FFFFFFF and 0xFF9AA4B2
Two stale values produce visually different glass and text colors from the spec. Serena search_for_pattern on both, replace with named tokens.
4
Increase zone pill to minimum 12sp with 40dp touch height
Zone status is safety-critical. Must be immediately legible at a glance during a run. ui/home/HomeScreen.kt · PulseHeroCard · ZonePill composable.
5
Raise TextTertiary to #6B6B73 for WCAG AA compliance on inactive nav labels
One token change in Color.kt fixes all nav labels and tertiary text globally. No screen-level changes needed.
6
Add gradient-tinted fill to selected PresetCard (GradientRed at 6% alpha)
Makes selected state clearly distinguishable from unselected at distance and in bright daylight. ui/setup/SetupScreen.kt · PresetCard.
