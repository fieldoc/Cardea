3. Day Picker — Clearer Legend & Interaction Hints
The current day picker has a tiny inline legend that doesn't distinguish the long-run star from regular run dots. Users don't discover that holding a day blocks it, or that tapping cycles through states. The proposed version has a 2×2 visual legend with descriptive text, a visible star on the long-run day, and clear interaction hints explaining tap/hold behavior.

Summary of Proposed Changes
Implementation Notes
Each proposal targets a specific UX gap identified from device screenshots. Changes are additive — no existing behavior is removed, only enriched.
Home Pre-Bootcamp
Feature showcase hero replaces minimal text card — HR zones, scheduling, voice coaching with icons
"Just run" strip — quick escape hatch for users who want to try a manual run before committing to bootcamp
Stat tiles kept but now feel less empty since the hero fills the visual weight above
Bootcamp Dashboard
Visual week strip shows completed/today/upcoming days with type labels and completion checkmarks
Session cards with icons — type-colored icons (green=easy, red=strides, blue=long) replace text-only Coming Up list
Phase timeline — segmented progress bar showing Base/Build/Peak/Taper with current phase highlighted
No dead space — content fills the screen without being redundant
Day Picker
2×2 legend grid with descriptive text for each state — no more guessing what "long" means
Visible star in both the legend and the day circle makes long-run day unmistakable
Interaction hints card explains tap cycling, hold-to-block, and long-run promotion with plain language
Subtitle updated from generic "Tap to toggle, hold to block" to "Choose which days work for running, and pick a day for your long run"
Voice Coaching Settings
Mode description cards below the segmented control explain what each mode includes
Feature tags (zone alerts, km splits, pace coaching, etc.) make the Minimal vs Full difference scannable
"Fine-tune Cues" label + helper text explains that sub-toggles only apply in Full mode
Active card highlighted with pink accent border — always shows what your current selection means
