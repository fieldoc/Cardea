CARDEA · HOME · HERO STATES

Three states the Home hero needs to know.

Today, Home only renders active bootcamp + new-user pitch. A marathon graduate sees the same screen as a fresh install. These three composables fix that — each tuned to a different emotional moment in the bootcamp lifecycle, all sharing one gradient hierarchy so they feel like the same product.

01 · GraduateHero 02 · ResumeCard 03 · NoBootcampCard

The three-gradient rule, applied to these heroes.

Tier 1 · Gradient

Triumph + identity

4-stop CardeaGradient on text only. One element per screen. Reserved for the moment that defines the screen.

→ GraduateHero headline · NoBootcampCard headline (existing) · PulseHero (active)
Tier 2 · Glass + White

Earned facts

White text on glass surface. Stats, progress, supporting metrics. No gradient — these are evidence, not the story.

→ Stats trio · Progress thread · Primary CTAs
Tier 3 · Glass + Secondary

Ambient context

Secondary or tertiary text on glass. Eyebrows, body copy, tertiary links. The quiet layer that lets Tier 1 breathe.

→ Pulse labels · Race chips · Freestyle / Manual-run links
Tier 1 · The triumph headline

Gradient text earns its keep

This is the only Home hero where the 4-stop CardeaGradient describes a moment, not a marketing line. Same BlendMode.SrcIn trick as NoBootcampCard:970. Italic period in GradientPink — small, but it lands like a stamp.

Eyebrow · earned tag

"GRADUATED · MARATHON"

Pulled from enrollment.goalType. Tertiary text + cyan pulse dot — same eyebrow primitive used everywhere else on Home. Right side carries a small amber trophy chip — the only warm pip on the screen, anchoring the laurel arc above.

Stats trio · earned, not flashy

Numbers in 30sp ExtraBold

Three glass tiles, monospace units in caps. Inner top-edge highlight (1px, 18% white) gives them a "minted" feel. Critically: no gradient — gradient on stats would steal from the headline and break Tier 2.

Primary CTA · Cardio Health

Wide row, not a button

Cardio Health is a destination, not an action — so it gets a row treatment with icon, title, and the "tier-aware" reassurance inline. EVERGREEN pip in cyan reinforces this is the calm-after-the-storm path. Hover-tints to cyan so it stands distinct from the warm-pink CTA gradient.

Race chips · secondary path

Four equal weights, no winner

Most graduates take Cardio Health (data point: most do). Race chips are quieter, equal-width — "if you want a new goal, here are the four shapes" — without bullying anyone toward another marathon block.

Freestyle escape hatch

Not "Manual Run" here

The wording matters. After graduation, "Just freestyle today" feels like a gift; "Manual Run" feels like a settings menu. Same destination as elsewhere — different door.

// ─────────────────────────────────────────────────────────────────────────────
// GraduateHero — Triumphant. The ONLY Home hero where the 4-stop gradient
// describes a moment ("Race-tier fitness, locked in") rather than a pitch.
// Tier-2 stats trio is glass + white, NOT gradient — earned numbers should
// read as evidence, not marketing. Trophy chip + amber laurel = the single
// warm beat that lifts gradient pink/cyan into "celebration" territory.
// ─────────────────────────────────────────────────────────────────────────────
@Composable
private fun GraduateHero(
    enrollment: BootcampEnrollmentEntity,
    weeksCompleted: Int,
    sessionsCompleted: Int,
    totalKm: Double,
    onChooseEvergreen: () -> Unit,
    onChooseRace: (RaceGoal) -> Unit,
    onFreestyle: () -> Unit,
    modifier: Modifier = Modifier,
) {
    val heroGradient = CardeaTheme.colors.gradient
    val goalLabel = remember(enrollment.goalType) {
        enrollment.goalType.name.replace("_", " ")
    }

    Box(
        modifier = modifier
            .fillMaxWidth()
            .padding(horizontal = HomeGutter)
            .clip(HomeCardRadius)                                // .clip BEFORE .background — children obey radius
            .border(1.dp, CardeaTheme.colors.glassBorder, HomeCardRadius)
            .background(
                Brush.verticalGradient(
                    colorStops = arrayOf(
                        0f to GradientPink.copy(alpha = 0.10f),
                        0.55f to GradientBlue.copy(alpha = 0.04f),
                        1f to CardeaTheme.colors.glassHighlight,
                    ),
                ),
            )
            .padding(HomeCardPadding + 4.dp),
    ) {
        Column(modifier = Modifier.fillMaxWidth()) {

            // Eyebrow + amber trophy chip ─ the single warm pip on screen.
            Row(
                modifier = Modifier.fillMaxWidth(),
                horizontalArrangement = Arrangement.SpaceBetween,
                verticalAlignment = Alignment.CenterVertically,
            ) {
                PulseLabel(text = "GRADUATED · $goalLabel")
                TrophyChip(weeks = weeksCompleted)
            }
            Spacer(Modifier.height(12.dp))

            // TIER 1 — gradient triumph headline.
            // Same compositingStrategy + drawWithContent + BlendMode.SrcIn
            // pattern used in NoBootcampCard:970-980. Do not reinvent.
            Text(
                text = "Race-tier fitness,\nlocked in.",
                style = MaterialTheme.typography.displaySmall.copy(
                    fontWeight = FontWeight.Black,
                    fontSize = 38.sp,
                    lineHeight = 40.sp,
                    letterSpacing = (-1.2).sp,
                ),
                color = CardeaTheme.colors.textPrimary,
                modifier = Modifier
                    .graphicsLayer { compositingStrategy = CompositingStrategy.Offscreen }
                    .drawWithContent {
                        drawContent()
                        drawRect(brush = heroGradient, blendMode = BlendMode.SrcIn)
                    },
            )
            Spacer(Modifier.height(8.dp))

            Text(
                text = "You finished your first marathon prep cycle. Every pace " +
                       "zone, every long run, banked. Here's where to take it next.",
                style = MaterialTheme.typography.bodyMedium.copy(fontSize = 13.5.sp, lineHeight = 19.sp),
                color = CardeaTheme.colors.textSecondary,
            )
            Spacer(Modifier.height(22.dp))

            // TIER 2 — stats trio. Glass + white. NO gradient.
            Row(
                modifier = Modifier.fillMaxWidth(),
                horizontalArrangement = Arrangement.spacedBy(8.dp),
            ) {
                StatTile(value = weeksCompleted.toString(), unit = "WEEKS", modifier = Modifier.weight(1f))
                StatTile(value = sessionsCompleted.toString(), unit = "SESSIONS", modifier = Modifier.weight(1f))
                StatTile(value = totalKm.toInt().toString(), unit = "KM", modifier = Modifier.weight(1f))
            }
            Spacer(Modifier.height(22.dp))

            PulseLabel(text = "CHOOSE YOUR NEXT PROGRAM")
            Spacer(Modifier.height(10.dp))

            // Primary path — Cardio Health (evergreen).
            EvergreenChoiceRow(onClick = onChooseEvergreen)
            Spacer(Modifier.height(10.dp))

            // Secondary path — pick a new race goal.
            Row(
                modifier = Modifier.fillMaxWidth(),
                horizontalArrangement = Arrangement.spacedBy(8.dp),
                verticalAlignment = Alignment.CenterVertically,
            ) {
                PulseLabel(text = "NEW RACE")
                RaceGoal.entries.forEach { goal ->
                    RaceChip(label = goal.shortLabel, onClick = { onChooseRace(goal) }, modifier = Modifier.weight(1f))
                }
            }
            Spacer(Modifier.height(18.dp))

            // Tertiary — freestyle escape hatch. Tertiary text, centered.
            TextButton(onClick = onFreestyle, modifier = Modifier.align(Alignment.CenterHorizontally)) {
                Text(
                    "Just freestyle today →",
                    color = CardeaTheme.colors.textTertiary,
                    style = MaterialTheme.typography.bodySmall,
                )
            }
        }
    }
}
02 · paused — quiet, no shame
No gradient title here

Plain serif headline

The pause was deliberate. A gradient would shout, and shouting is exactly the wrong tone for "you stepped away on purpose." Headline is Playfair white on glass — friendly, calm, unambiguously not a graduation moment.

Eyebrow · "PAUSED · WEEK 7"

Where, not why

Tells the user where they paused (week N), not why (we don't know, and shouldn't pretend). No red, no warning chrome — pause is a neutral state. cyan dot stays consistent with active and graduated cards.

Progress thread · breadcrumb

"15 of 26 sessions"

One quiet stat. White-on-glass progress bar (no gradient — that's reserved). The right edge sits where they paused. Reads as a bookmark, not a debt: "your place is held."

// ─────────────────────────────────────────────────────────────────────────────
// ResumeCard — Warm, not nagging. Pause is neutral, never a problem.
// No gradient title — gradient ink would feel like marketing trying to win
// the user back. The single Tier-1 element is the CTA gradient itself.
// Outlined "Start different program" preserves dignity for users who paused
// because the goal was wrong (not because life got busy).
// ─────────────────────────────────────────────────────────────────────────────
@Composable
private fun ResumeCard(
    enrollment: BootcampEnrollmentEntity,
    sessionsDone: Int,
    sessionsTotal: Int,
    pausedAtWeek: Int,
    totalWeeks: Int,
    onResume: () -> Unit,
    onSwitchProgram: () -> Unit,
    onManualRun: () -> Unit,
    modifier: Modifier = Modifier,
) {
    val goalLabel = remember(enrollment.goalType) {
        enrollment.goalType.name.replace("_", " ").lowercase()
    }

    Box(
        modifier = modifier
            .fillMaxWidth()
            .padding(horizontal = HomeGutter)
            .clip(HomeCardRadius)
            .border(1.dp, CardeaTheme.colors.glassBorder, HomeCardRadius)
            .background(CardeaTheme.colors.glassHighlight, HomeCardRadius)
            .padding(HomeCardPadding),
    ) {
        // Pause glyph — tiny, top-right. Two glass bars. No more.
        PauseGlyph(modifier = Modifier.align(Alignment.TopEnd))

        Column(modifier = Modifier.fillMaxWidth()) {
            PulseLabel(text = "PAUSED · WEEK $pausedAtWeek")
            Spacer(Modifier.height(12.dp))

            // NO gradient. Plain serif white-on-glass.
            Text(
                text = "Ready when\nyou are.",
                style = MaterialTheme.typography.headlineLarge.copy(
                    fontWeight = FontWeight.SemiBold,
                    fontSize = 30.sp,
                    lineHeight = 32.sp,
                    letterSpacing = (-0.5).sp,
                ),
                color = CardeaTheme.colors.textPrimary,
            )
            Spacer(Modifier.height(8.dp))

            Text(
                text = buildAnnotatedString {
                    append("You had ")
                    withStyle(SpanStyle(color = CardeaTheme.colors.textPrimary, fontWeight = FontWeight.SemiBold)) {
                        append("$sessionsDone of $sessionsTotal")
                    }
                    append(" sessions complete in the ")
                    withStyle(SpanStyle(color = CardeaTheme.colors.textPrimary, fontWeight = FontWeight.SemiBold)) {
                        append(goalLabel)
                    }
                    append(" program. Your place is held — pick up exactly where you left off.")
                },
                style = MaterialTheme.typography.bodyMedium.copy(fontSize = 13.5.sp, lineHeight = 19.sp),
                color = CardeaTheme.colors.textSecondary,
            )
            Spacer(Modifier.height(22.dp))

            ProgressThread(
                sessionsDone = sessionsDone,
                sessionsTotal = sessionsTotal,
                pausedAtWeek = pausedAtWeek,
                totalWeeks = totalWeeks,
            )
            Spacer(Modifier.height(22.dp))

            // TIER 1 — the only gradient on this card. Buttons-only CTA gradient.
            CardeaButton(
                text = "Resume training",
                onClick = onResume,
                leadingIcon = Icons.Default.PlayArrow,
                innerPadding = PaddingValues(horizontal = 24.dp, vertical = 14.dp),
                modifier = Modifier.fillMaxWidth(),
            )
            Spacer(Modifier.height(10.dp))

            OutlinedButton(
                onClick = onSwitchProgram,
                modifier = Modifier.fillMaxWidth().height(44.dp),
                shape = RoundedCornerShape(14.dp),
                border = BorderStroke(1.dp, CardeaTheme.colors.glassBorder),
            ) {
                Text(
                    "Start a different program",
                    color = CardeaTheme.colors.textPrimary,                  // avoid M3 primary leak
                    style = MaterialTheme.typography.bodyMedium,
                )
            }
            Spacer(Modifier.height(14.dp))

            TextButton(onClick = onManualRun, modifier = Modifier.align(Alignment.CenterHorizontally)) {
                Text(
                    "Manual run →",
                    color = CardeaTheme.colors.textTertiary,
                    style = MaterialTheme.typography.bodySmall,
                )
            }
        }
    }
}
03 · refresh — don't rebuild
Refresh, not rebuild

Existing structure preserved

Same gradient title pattern (line 970-980), same FeatureItem rows, same "Set Up Bootcamp" CTA. What changed: tighter copy, sharper feature trio, feature rows now grouped in their own glass container so they read as a system, not three loose icons.

Subtitle · before/after

Cut "around your life, fitness…"

Old: "Cardea builds an adaptive plan around your life, fitness, and heart rate data." Three concepts in one sentence is one too many — "adaptive" is the verb, the rest belongs in features. New: "An adaptive plan that knows your week, your heart, and your pace."

Feature trio · the right three

HR · Schedule · Voice

Current trio is correct: HR-zone coaching is the headline differentiator, life-aware scheduling is what keeps you in the program, voice cues are the in-run delivery. Don't swap them. Tighten the descriptions.
