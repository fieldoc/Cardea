TIER 1 Goal: 3/4 The ONLY gradient number on screen. Subtle warm border glow separates it from everything else. This is the "do I run today?" answer — it earns the color.
TIER 1 Streak: 7 Plain white — no gradient. Still large, still prominent, but it doesn't compete with Goal. The streak reinforces motivation without demanding action.
TIER 2 Bootcamp: 42% Glass card with a muted gradient bar (0.7 opacity) and tiny ring. The percentage is secondary text color, not gradient. Progress is visible but doesn't scream.
TIER 3 Volume: 12.4/15km Nearly invisible — ghost-colored text, 3px bars with 10% white fill. You can see it if you look. You don't notice it if you don't. Reference data, not actionable.
TIER 3 Coaching: italic whisper No card, no border, no bold text. Just an italic line with a tiny faded ECG icon. The coaching is ambient — it's there when you need it, invisible when you don't.
Design Principle The Cardea gradient appears in exactly 3 places below the hero: the Goal number, the CTA button, and the bootcamp progress bar (muted). Everything else is white, gray, or invisible. Restraint makes the gradient meaningful.
